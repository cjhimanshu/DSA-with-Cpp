# DSA with C++

This repository contains C++ solutions for data structures and algorithms problems.

## Contents

- Problem solutions collected in `ds.cpp`
- Practice work for common coding interview and DSA questions

## How to Use

Compile the C++ file with any standard-compliant compiler, for example:

```bash
g++ ds.cpp -o ds
./ds
```

## Notes

- Solutions are being added incrementally
- The repository is intended for learning and practice
